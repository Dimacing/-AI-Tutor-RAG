# Demo-скрипт

1) Показать источники
   - Откройте `data/sources.json` и упомяните, какие материалы индексируются.

2) Построить индекс
   - Через UI: нажать «Создать индекс» для режима Cloud docs.
   - Или через CLI: `python -m backend.rag.index --sources data/sources.json`.

3) Запустить бэкенд и UI
   - API: `uvicorn backend.main:app --reload`
   - Веб-UI: `http://localhost:8000`
   - (Опционально) Streamlit: `streamlit run ui/streamlit_app.py`

4) Демонстрационный вопрос
   - «Я прочитал про Object Storage (S3), но не понимаю отличие от блочного хранилища. Объясни просто и задай вопрос для самопроверки.»

5) Показать результат
   - Ответ с источниками
   - Самопроверка
   - Проверочный тест

6) Проверка безопасности
   - Спросите что-то с персональными данными, чтобы показать предупреждение.
